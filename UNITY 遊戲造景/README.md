# UNITY 遊戲造景
* 此作品為修習多媒體技術與應用所製作，運用到了UNITY的基本操作及程式設計的邏輯基礎，使用遊戲開發工具UNITY及C#語言開發，製作出可以操控角色移動、改變視角的小遊戲。
