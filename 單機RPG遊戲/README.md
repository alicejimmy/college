# 單機RPG遊戲
* 此作品為修習軟體工程時所製作，運用到了UNITY的基本操作及程式設計的邏輯基礎，使用遊戲開發工具UNITY及C#語言開發，製作出可以實際遊玩，操控角色體驗遊戲劇情的單機RPG遊戲。
* 直接執行game_test\FT_0107_1808.exe就可運作
