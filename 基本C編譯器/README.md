# 基本C編譯器
* 此作品為修習編譯器所製作，運用到了編譯器的基本觀念，使用C語言開發，撰寫出可以編譯基本C語言的自製編譯器，會將輸入的高階語言.c檔先轉成機器語言.asm檔，再轉換成執行檔並執行程式。
