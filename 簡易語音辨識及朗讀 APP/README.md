# 簡易語音辨識及朗讀APP

* 此作品為修習行動裝置應用程式開發所製作，運用到了SpeechToText及TextToSpeech，使用APP開發工具Android Studio及JAVA的語言開發，製作出可以文字、語音雙向轉換的簡易APP。
